# 图标文件说明

请将您的应用图标文件放置在此目录下：

1. **icon-192.png** - 192x192 像素的 PNG 图标
2. **icon-512.png** - 512x512 像素的 PNG 图标

## 图标要求：
- 格式：PNG
- 尺寸：192x192 和 512x512
- 背景：建议透明或与主题色一致
- 内容：可以使用您提供的蓝色纸飞机图标

## 快速生成图标：
如果您有原始图标文件，可以使用在线工具生成不同尺寸：
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## 临时方案：
如果暂时没有图标文件，可以：
1. 使用任何 192x192 和 512x512 的图片文件
2. 重命名为 icon-192.png 和 icon-512.png
3. 放置在此 public 目录下
